How it works

Five phases, one approval gate, one verify-iterate loop. The verify pass runs 11 defect classes in parallel — five visual (overflow, empty placeholder, layout mismatch, brand violation, density) and six rhetorical (claim-title, one-idea, bullet-dump, audience-mismatch, red-line-break, curse-of-knowledge). The deck only ships when all eleven are green.
